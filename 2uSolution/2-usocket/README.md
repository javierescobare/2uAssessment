# 2usocket

Web socket project for real-time communication

## Installation 🔧

Install project dependencies using:

```bash
npm install
```

## Usage 🚀

You can just run:

```bash
npm start
```

This project is already deployed and hosted on [Azure](https://2usocket.azurewebsites.net).

## Tools used 🛠️

-   Socket.io
-   Axios
